## Adding New Designs

To add a new design to the configuration:

1. **Extract the File ID from the Figma URL:**

   From: `https://www.figma.com/design/[FILE_ID]/[design-name]`

   Example: `https://www.figma.com/design/xGaucKjUbmEsgDqxjVvnU5/105-Forms-design`

   File ID: `xGaucKjUbmEsgDqxjVvnU5`

2. **Update `config.json`** (or `config.template.json` as a template):

```json
"designs": {
  "Your Design Name": {
    "fileId": "extracted-file-id",
    "url": "full-figma-url",
    "description": "Brief description of the design"
  }
}
```

3. **Update this file** with the new design reference for easy lookup.
